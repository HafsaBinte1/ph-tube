The scope of a var variable is functional scope. The scope of a let variable is block scope. It can be reassign in same scope. The scope of a const variable is block scope.

If the user is not logged in, you could assign the value null to the user variable to indicate that there is no logged-in user. On the other hand, undefined is typically used when you want to check if a variable has been assigned a value or not.

A REST API is an API that conforms to the design principles of the REST, or representational state transfer architectural style. For this reason, REST APIs are sometimes referred to RESTful APIs.
